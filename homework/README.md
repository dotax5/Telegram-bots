### Для запуска бота нужно подправить строки в файле ```data.env```:
```python
token=TOKEN
TELEGRAM_TOKEN=TOKEN
CHAT_ID=YOUR_TG_ID # можно поставить свой id (кому бот присылает дз)
PHONE_NUMBER=PHONE_NUMBER
PASSWORD=PASSWORD_GOSUSLUGI
```
